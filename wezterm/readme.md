# Wezterm


## キーバインド

weztermでの操作はvimとの競合を避けるため,`<Leader> + key`によって行う.
なお,`<Leader>`は`<CTRL> + a`とした.


|操作|キーバインド|
|:-|:-|
|垂直分割|`<Leader> + |`|
|水平分割|`<Leader> + %`|
|ペイン選択|`<Leader> + p`|
|ペインを閉じる|`<Leader> + w`|
|新規タブ|`<Leader> + t`|
|タブを閉じる|`<Leader> + q`|
|次のタブ|`<Leader> + n`|
|前のタブ|`<Leader> + b`|
|フルスクリーン切り替え|`<Leader> + f`|
|ペインのリサイズ|`<Leader> + hkjl`|
